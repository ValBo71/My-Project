Целата на този тест е проверка на правилното изпълнение на набор от ръчно проведени тестове
които се използват за регресивен тест. 
Използвайки възможностите на API, се проверява дали въвеените стойности, се отразяват правилно в отговора на полуяен от API.

Тук съм качил само част от тестовете, с цел да не нарушавам конфеднциалността.


Как работи тестът

1. С помощта на XML Schemas се създават случаи в iMX.

2. Използвайки скрипт в Excell получавам всички необходими за теста данни от XML схемите. 
    Скриптът създава *.csv файл
3. Използвам RUN функциите на Postman за да заредя всички данни от *.csv файла в collection variables.
4. Със заредените вече данни, могат да се изпълняват останалите сетове от тестове както общо, така и по единично.
